# GesturEd

> **Learning you can touch.**

Traditional education confines learning to passive consumption — students read, watch, and forget. There is no intuitive way for learners to *physically interact* with abstract concepts, leaving critical gaps in understanding and retention. **GesturEd** bridges this by transforming hand gestures into a learning interface, making education truly hands-on.

---

## What is GesturEd?

GesturEd is an **AI-Native EdTech platform** that uses real-time hand gesture recognition to let students interact with and manipulate educational concepts visually — turning abstract ideas into tangible, explorable experiences.

No mouse. No keyboard. Just your hands.

---

## 🚀 Features

- **Gesture-Based Interaction** — Control, manipulate, and explore concepts using natural hand movements via webcam
- **Real-Time Visualization** — Instantly see how changing a variable or parameter affects the concept
- **Multi-Subject Support** — Mathematics, Physics, Chemistry, Biology, and more
- **Active Learning Engine** — Every session is interactive, not passive
- **Browser-Based** — No installation required; works directly in the browser

---

## 🧩 How It Works

```
Student opens topic
        ↓
Webcam captures hand gestures in real-time
        ↓
MediaPipe / gesture model interprets the gesture
        ↓
Interactive visualization responds instantly
        ↓
Student explores all variations of the concept
        ↓
Deeper understanding + stronger retention
```
